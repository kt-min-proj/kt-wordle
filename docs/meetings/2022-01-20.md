# 회의 내용
## 화면 및 기능
1. 로그인/회원가입
2. 워들 게임 -> 내 점수 + 캘린더 (그날 랭크 1-5 혹은 1-10위까지 출력), 로그인 한 사람만 게임 가능

- admin유저가 그날 값 입력




## DB
사용자 / DayRank / 순위
1. User: 이름, ID, 비번
2. DayRank: 맞춘 횟수, 순위, 시간, 날짜, 점수, FK(user)
3. Rank: 날짜, 정답, FK(1~5 or 10) -> 여쭤보기	
	- Rank 따로 빼는게 좋을지, DayRank 와  합치는게 좋을지 튜터님께 여쭤보기
